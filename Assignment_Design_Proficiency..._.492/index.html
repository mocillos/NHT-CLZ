Introduction:

  • In this activity, we will be testing your ability to use the VistaStudio to recreate a design using only shapes. 
  • This activity will be graded.

Instructions:

  • Recreate the design following the steps listed below.

Time to complete activity:

60 mins

Steps:

1.    Go to www.vistaprint.com and log into your Vistaprint account.

2.    Select a standard postcard template.

3.    Customize the template so that it looks exactly as the postcard shown below.

        o    You must use the same colors, shapes, information and font.

4.    Make sure you save the design to your account.

5.    Order a Proof of your design and save it to your PC.

6.    Once you have completed designing the postcard, return to the CARE Learning Zone. 

7.    Submit the Proof you created below.

8.    Click the Submit all and finish button to submit your work. This is necessary in order for the activity to be marked as complete.
